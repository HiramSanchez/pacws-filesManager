## **Setup**
1. Clone the repo
```bash
git clone https://github.com/your-user/your-repo.git
```

2. Install dependencies
```bash
npm install
```

3. Run the project
- backend first
```bash
../backend> node server.js
```
- frontend after
```bash
../frontend> npm run dev
```
